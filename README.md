# NotifyX

> A production-grade Event-Driven Notification Platform built using Java, Spring Boot, Kafka, PostgreSQL, Redis, Docker and Kubernetes.

---

## 🚀 Tech Stack

- Java 21
- Spring Boot
- Maven
- Apache Kafka
- PostgreSQL
- Redis
- Docker
- Kubernetes
- Prometheus
- Grafana
- OpenTelemetry
